CSS Interview Questions


Q1-What is CSS and why is it used?
Ans 1

CSS stands for cascading style sheet, it is used to describe the presentation i.e. layout, colors, fonts and spacing of HTML documents. It seprates content from design and making it easier to maintain and style web pages consistently across a website.


Q2-What are the different types of CSS?
Ans 2

There are three types of CSS:
1. Inline CSS: Styles are applied directly to HTML elements using the style attribute.
2. Internal CSS: Styles are defined within a <style> tag inside the <head> section of the HTML document.
3. External CSS: Styles are written in a separate CSS file and linked to the HTML document using the <link> tag.


Q3 - How can you include CSS in an HTML document?
Ans 3

We can include CSS in our HTML file in 3 diffrent ways:

  • Inline (style attribute in HTML tags)
  • Internal (style tag inside the head)
  • External (link tag referring to a .css file)

Q4 - What is specificity in CSS?
Ans 4

Specificity determines which CSS rule is applied when multiple rules target the same element.
It's calculated based on the following priority:
Inline styles > IDs > Classes/Pseudo-classes/Attributes > Elements

Example : 

            #id { }        specificity: 100 
            .class { }     specificity: 10 
            div { }        specificity: 1

Q5 - What is the difference between em, rem, px, and %?
Ans 5
  • px – Absolute unit (e.g., 16px)
  • em – Relative to the font-size of the parent element
  • rem – Relative to the root (html) element's font-size
  • % – Relative to the parent element’s value (often used in width, height, etc.)

Q6 - What is the difference between id and class in CSS?
Ans 6
Class ID
